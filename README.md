# Demo Presentation Repository

This folder contains the shared demo/presentation site for the innovation project.

## Folder Layout

- `index.html` - Cross-module demo landing page.
- `assets/` - Images and static assets used by the demo page.
- `docs/` - Local proposal and presentation reference documents. This folder is excluded from Git.

## Contribution Flow

Each module team can maintain its own module repository. When a team wants their module included in the shared demo, they can open a pull request here that updates `index.html` with the module link, status, and demo details.
